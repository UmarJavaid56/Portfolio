About Me

Projects

16-bit Hack Computer

This repository contains my implementation of the Hack computer as described in "The Elements of Computing Systems" by Noam Nisan and Shimon Schocken. The project includes the complete hardware architecture of the Hack platform, from the ALU and CPU to the assembler.
Pong in Java

My own version of the iconic Pong Game. To play the game, you can download the code from my GitHub repository.
ComfortClick

A touch-sensitive, ergonomic television remote designed to minimize button presses and enhance accessibility. It features large buttons and a split-case design for easy attachment to wheelchairs.
Spatial Mapping and Data Acquisition

The LIDAR-based 3D scanning system uses a Time-of-Flight sensor, stepper motor, and ARM Cortex-M4F microcontroller for precise indoor mapping.
Nandland GoBoard Projects

Using the GoBoard hardware platform, I've designed, built, and tested various digital circuits. This hands-on experience has significantly enhanced my understanding of digital electronics and FPGA programming.

Automated Waste Management

Designed and programmed robots to identify, pick up, and sort waste by weight, color, and material, ensuring accurate disposal in appropriate bins.
